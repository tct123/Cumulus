KRT2 radio pairing

Cumulus can connect a radio device of type KRT2 via the WiFi interface WiFi-KRT2 and can transmit a frequency to be set on the radio. Here I will describe the usage of an XCVario device for this purpose. The XCVario device is working as a WiFi server and router for Cumulus. The KRT2 device is connected to the XCVario device via the serial interface S2. The serial interface S2 of the XCVario has to be configured in the following way:

9600 baud, 8 bits, without parity

The following IP parameters are to set at the GPS configuration page at the item WiFi-KRT2:

192.168.4.1:8882

In different windows, showing radio frequencies, you can see a button KRT2. Pushing this button will open a new window. In a table all frequencies related to the object, are to see. Every table entry has four columns.

Frequency Active Standby Call sign
Frequency to setup Setup as Active Setup as Standby Radio call sign


If you press the column Active or Standby the displayed frequency is sent to the KRT2 radio device according to the chosen attribute.

A LED, located at the right lower side of the window, shows with green light an active connection to the radio device. A none active connection is signaled by the LED with red light.

Circle Button

Pressing the circle button will swap the active and standby frequency at the radio.
